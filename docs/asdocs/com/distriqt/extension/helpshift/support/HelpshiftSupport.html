clearBreadCrumbs()method
public function clearBreadCrumbs():void

Clears Breadcrumbs list. Breadcrumbs list stores upto 100 latest actions. You'll receive those in every Issue. But if for reason you want to clear previous messages (On app load, for eg), you can do that by calling following function.

isConversationActive()method 
public function isConversationActive():Boolean

You can check if the user has any active Conversation on the device. As soon as an end user opens the conversation screen, they will see a greeting message, and the conversation is considered active.

Returns
Boolean — true if there is an active conversation

showFAQSection()method 
public function showFAQSection(sectionPublishId:String, config:ApiConfig = null):void

Show a particular section in the FAQs. This will show a FAQ section view with list of questions in that section. The search inside this view will be limited to the specified section. You can specify a section using publish ID of that section.

Parameters

sectionPublishId:String — the FAQ section publish-id
 
config:ApiConfig (default = null) — Additional config for the view
